# Rescate en la Selva

Videojuego 2D desarrollado con Phaser.js y JavaScript.

## Historia

Leo y Valeria se encontraban conversando tranquilamente sobre los examenes frente a la FIS. De repente aparece el Dr. Mortis, quien dice que Valeria es justo la pieza que necesita para completar su experimento. El doctor atrapa a Valeria y se la lleva hacia su laboratorio oculto en la selva. Leo queda destrozado, pero decide no rendirse y entra a la selva para rescatarla.

## Objetivo

Derrotar a tres jefes para salvar a Valeria:

1. Murk, el murcielago venenoso.
2. Brakus, el cui salvaje mutado.
3. Dr. Mortis, el cientifico loco.

## Controles

- A / Flecha izquierda: mover a la izquierda.
- D / Flecha derecha: mover a la derecha.
- W / Espacio / Flecha arriba: saltar.
- J o X: atacar con machete.
- P: pausar.

## Mecanicas

- Leo aparece al lado izquierdo y es el unico personaje que camina.
- Los jefes aparecen al lado derecho.
- Murk solo sube y baja, y lanza tres bolas de veneno cada 5 segundos.
- Brakus se queda quieto y lanza fuego cada 5 segundos.
- Dr. Mortis se queda quieto, dispara un rayo cada 5 segundos y hace caer tres rocas.
- El juego usa barras de vida, puntaje, pausa, mute, game over, victoria y localStorage.

## Ejecucion

```bash
npm install
npm run dev
```

Luego abrir la URL local que aparece en la terminal, normalmente:

```text
http://localhost:5173/
```
